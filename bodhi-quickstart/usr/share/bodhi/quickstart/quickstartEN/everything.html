Run Everything


[click on any image to enlarge]

Everything is an Moksha module which, as its name implies, allows you to run (almost) everything. Want to get to the Favorite Applications dialog box to set up your Favorites Menu? Then just type fav into Everything and hit Enter. Want to launch Terminology? Then enter ter into Everything. This brief guide can only describe the basic features of Everything; for more information follow the links on the Resources page at the end of this QuickStart Guide.

You can Run Everything by clicking its gadget on the shelf, or by pressing the key bindings Alt+Esc or Ctrl+Alt+Space, or from Main Menu > Run Everything. It will close automatically after performing the requested action, otherwise close it by pressing Esc or clicking outside its window.
